## About This Branch

This is a modified release of **VSecM** SDK to work with Go v1.18 or above. 

For the SDK versions that uses a higher version of go, please refer to the
[VMware Secrets Manager SDK on the `main` 
branch](https://github.com/vmware-tanzu/secrets-manager/tree/main/sdk).
